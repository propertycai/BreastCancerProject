# 🏥 乳腺癌辅助诊疗系统

一个基于机器学习的乳腺癌诊断和生存预测辅助决策系统，具有美观的现代化界面和完整的移动端适配。

## ✨ 特性

### 🎯 核心功能
- **乳腺癌诊断预测**：基于细胞学特征参数预测肿瘤良恶性
- **乳腺癌生存预测**：基于患者临床信息和分期数据预测生存期
- **结果保存**：支持将预测结果保存为JSON文件
- **实时验证**：表单输入实时验证和反馈

### 🎨 界面设计
- **现代化UI**：渐变背景、霓虹发光效果、粒子动画
- **响应式布局**：完美适配桌面、平板、手机
- **左侧垂直导航**（所有设备）：宽度自适应（220px → 80px → 70px → 65px）
- **按钮垂直排列**：所有设备都保持垂直导航布局
- **流畅动画**：60fps动画效果，优化的性能

### 📱 移动端适配
- ✅ **完全响应式**：支持从360px到4K所有屏幕
- ✅ **触摸友好**：44px最小触摸目标（WCAG标准）
- ✅ **iOS优化**：防止自动缩放、平滑滚动
- ✅ **Android优化**：完美支持Chrome Mobile
- ✅ **横屏支持**：自动调整布局

## 🚀 快速开始

### 环境要求
- Python 3.x（内置HTTP服务器）
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 启动服务器

```bash
# 方法1：使用Python (推荐)
cd /Users/property/Code/web/BreastCancerProject
python3 -m http.server 8000

```

### 访问地址
- **本地访问**：http://localhost:8000
- **局域网访问**：http://[你的IP]:8000

查找IP地址：
```bash

ifconfig | grep "inet "

```

```bash

lsof -ti:8000 | xargs kill
```

## 🎯 使用说明

### 1. 系统简介
- 查看系统功能和使用说明
- 了解主要模块
- 阅读重要提示

### 2. 乳腺癌诊断预测
1. 填写9个细胞学特征参数（范围0-10）
   - 肿瘤厚度
   - 细胞大小均匀性
   - 细胞形状均匀性
   - 边缘粘附力
   - 单上皮细胞大小
   - 裸核
   - 染色质的颜色
   - 核仁正常情况
   - 有丝分裂情况
2. 点击"确认上传"进行预测
3. 查看预测结果（良性/恶性）
4. 可选择"保存结果"

### 3. 乳腺癌生存预测
1. 填写患者基本信息
   - 年龄、种族、婚姻状况
   - 家庭收入、居住区域
2. 填写临床分期数据
   - AJCC分期
   - TNM分期
3. 填写治疗方案
   - 手术、放疗、化疗
4. 点击"确认上传"进行预测
5. 查看预测结果
6. 可选择"保存结果"
